Current Workflow

Work on small task and create PR for other dev

CircleCI - Pros

  • Bootstrapped startup price was right
  • Secrets not in code and entered on CircleCI site
  • CircleCI allows you to SSH into the build machine, which is cool

CircleCI - Cons

  • Free tier has one worker
  • Heavy on the configuraton side, e.g. node_modules

Spawned Environment

  • Once per PR if successful spins up complete environment on external server
  • Presents user with URL and SSH details
  • Released to public soon

CircleCI Configuration

  • Machine > Services > docker
  • Dependencies - Docker Compose, Docker pulls, Configuration file, Docker save, Docker Compose build, Ran npm install in several folders
  • Dependencies - Ignore cache_directories (node_modules)

CircleCI great once setup
